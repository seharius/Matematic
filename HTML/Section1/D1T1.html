Wykres funkcji wymiernej f(x) = a/x

Wykres funkcji f(x) = a x określonej dla X ∈ R \ {0} ,gdzie a ≠ 0 nazywamy hiperbolą.
Hiperbola składa się z dwóch gałęzi. Ma asymptotę pionową o równaniu x = 0 i asymptotę poziomą o równaniu y = 0

a ≻ 0

Funkcja f(x) = a x , gdzie X ≠ 0 oraz a ≻ 0 jest malejąca w każdym z przedziałów
(-∞;0) oraz (0;∞).
